# Floating Presentation Timer

A lightweight, "Always on Top" countdown timer for presentations and Q&A sessions.

## Features
- **Floating & Borderless**: Minimalist design that stays on top of your slides.
- **Draggable**: Click and drag anywhere on the window to position it.
- **Presets**: Quick buttons for 8-minute (Presentation) and 5-minute (Q&A) sessions.
- **Visual Alerts**: Time turns red in the last 60 seconds and flashes when time is up.

## How to Run
Ensure you have Python installed, then run:
```bash
python timer.py
```

## Controls
- **8 Min / 5 Min**: Reset the timer to the preset.
- **Start / Pause**: Start or pause the countdown.
- **Reset**: Reset to the beginning of the current session.
- **✕**: Close the application.
